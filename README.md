# MyRoutine

## Over het project

My Routine is een interactieve fitnesswebsite die is ontworpen om je te helpen structuur aan te brengen, gemotiveerd te blijven en je persoonlijke fitnessdoelen bij te houden. Het platform helpt zowel beginners als meer ervaren gebruikers bij het opstellen en volhouden van een effectief trainingsschema.

## Live Demo

Bekijk de website hier => https://hellokittyuncensored.github.io/MyRoutine/

## Pagina's

| Pagina | Beschrijving |
|--------|-------------|
| **Home** | Startpagina met een overzicht van de website en animaties |
| **Training** | Overzicht van push, pull en leg workouts |
| **Planner** | Plan je wekelijkse trainingsschema |
| **Progress** | Volg je vooruitgang via grafieken met D3.js |
| **Webmentions** | Overzicht van alle externe vermeldingen van de website |

## Technologieën

- HTML
- CSS
- JavaScript
- GSAP (animaties)
- D3.js (datavisualisatie)
- RSS Feed
- Webmentions

## Functies

- Interactieve animaties met GSAP en ScrollTrigger
- Dynamische datavisualisatie met D3.js
- RSS feed voor updates
- Webmentions overzichtspagina
- Responsief ontwerp
